# wasd-keyset-starwars-appleii

Files and templates for a custom Star Wars / Apple II themed keyset for [WASD keyboards](https://www.wasdkeyboards.com/). I'm sorry this repo is such a mess. :D

## What's in here

- `SVGs/` — keycap artwork in SVG format, ready to upload to WASD's customizer
- `Icons/` — individual icon assets used in the keycap designs
- `Fonts/` — fonts used in the designs

## Using with WASD

1. Download the SVGs
2. Upload to [WASD's custom layout tool](https://www.wasdkeyboards.com/products/mechanical-keyboard-accessories/wasd-v3-87-key-custom-mechanical-keyboard.html)
3. Adjust sizing as needed for your keyboard model
